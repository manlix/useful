Common way:

`[CTRL]` **+** `[SHIFT]` **+** `[U]` **+** `[UNICODE_CODE]` **+** `[ENTER]`

Example: long dash `—` instead standard minus `-`

`[CTRL]` **+** `[SHIFT]` **+** `[U]` **+** `2014` **+** `[ENTER]`

Result: `—`

For Windows 11 (for typing 'long dash'):

`2014` + `[ALT]` + `[X]`

or

`U+2014` + `[ALT]` + `[X]`
